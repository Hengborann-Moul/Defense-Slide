We train four independent XGBoost classifiers, one per affective state, boredom, engagement, confusion and frustration. The justification for four independent models rather than a shared-weight multi-task head is that the per-state feature dependencies differ enough that shared weights hurt minority states. Each model uses max_depth 8, learning_rate 0.05, 500 estimators, subsample 0.8, colsample_bytree 0.8, min_child_weight 3, gamma 0.1, reg_alpha 0.1, reg_lambda 1.0, multi:softprob objective, and 50-round early stopping, all tuned via 5-fold cross-validation on the training fold. Imbalance is attacked at three levels: data via SMOTE with k=5 neighbors applied to the training split only; algorithmic via inverse-frequency class weights w_c equals N over K times N_c; and loss via Focal Loss with alpha 0.25 and gamma 2.0, originally proposed for object detection and adapted here as a custom XGBoost objective with explicit gradient and Hessian. Finally, isotonic regression calibrates the tree-ensemble probabilities, and a threshold search in 0.2 to 0.8 maximizes Macro F1.
